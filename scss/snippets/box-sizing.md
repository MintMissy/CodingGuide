# Box Sizing Snippets

## Generic

```scss
box-sizing: border-box; // boz

height: 1px; // hei
min-height: 1px; // mih
max-height: 1px; // mah

width: 0; // wid
width: auto; // wida
min-width: 1px; // miw
max-width: 1px; // maw
```

## Padding

```scss
padding: 0; // pad
padding-top: 0; // padt
padding-right: 0; // padr
padding-bottom: 0; // padb
padding-left: 0; // padl
```

## Margin

```scss
margin: 0; // mar
margin: 0 auto; // mara
margin-top: 0; // mart
margin-right: 0; // marr
margin-bottom: 0; // marb
margin-left: 0; // marl
```

## Border

```scss
border: 1px solid #eee; /* bor */
border: none; /* born */

border-top: 1px solid #eee; /* bort */
border-right: 1px solid #eee; /* borr */
border-bottom: 1px solid #eee; /* borb */
border-left: 1px solid #eee; /* borl */

border-width: 1px; /* borw */
border-style: solid; /* bors */
border-color: #eee; /* borc */

border-radius: 2px; /* br */
```
