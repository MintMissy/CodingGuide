# Position Snippets

## Position Types
```scss
position : relative; // pos
position : absolute; // posa
position : fixed; // posf
position : relative; // posr
position : sticky; // poss
```

## Insets
```scss
top : 0; // top
right : 0; // rig
bottom : 0; // bot
left : 0; // lef
```

## Z-Index
```scss
z-index : -1; // zi
```

## Vertical Align
```scss
vertical-align : baseline; // va
vertical-align : bottom; // vab
vertical-align : middle; // vam
vertical-align : top; // vat
```
