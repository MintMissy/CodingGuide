# Animation Snippets

```scss
animation : name 1s linear; /* ani */

animation-name : name; /* anin */
animation-duration : 1s; /* anidu */
animation-timing-function: linear; /* anitf */

animation-delay : 1s; /* anide */
animation-direction : alternate; /* anidi */
animation-fill-mode : forwards; /* anifm */
animation-iteration-count: infinite; /* aniic */
animation-play-state : paused; /* anips */

@keyframes name {}; // key
```
